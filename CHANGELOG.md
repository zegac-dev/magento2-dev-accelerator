# Changelog

All notable changes to the ZegacLabs Dev Accelerator module will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-11-24

### Added
- **Template Debugger Plugin** - Injects HTML comments showing template file paths in rendered output
- **Container Debugger Plugin** - Marks layout containers with HTML comments for structure visualization
- **Layout Handles Logger Plugin** - Logs page layout handles to browser console
- Admin configuration panel under Stores > Configuration > ZegacLabs > Dev Accelerator
- Store-scoped configuration support
- Helper class for centralized config management
- Full documentation (README.md)
- PHP 8.2, 8.3, 8.4 support
- Magento 2.4.8+ compatibility
